About this app

LodSight-visualizer

This software visualizes stored summarizations of linked datasets created by LodSight-summarizer. It obtains them from LodSight-graph-creator. Its source code is available on GitHub as LodSight-visualizer .

LodSight-summarizer

LodSight-summarizer is an experimental implementation of a dataset summarization algorithm which creates a (data)type-property graph summarization of an LOD dataset by accessing its SPARQL endpoint and stores the results into a relational db for offline viewing. Look at the source code.

LodSight-graph-creator

LodSight-visualizer gets the summarization through a simple php application LodSight-graph-creator. It returns selected summarization as an easily displayable json object.
